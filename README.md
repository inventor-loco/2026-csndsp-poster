Presentation Format and Template





To enhance interactions and engagement, all accepted papers will be presented using a combination of oral and poster presentations. Each oral presentation will be a 3-minute lightning talk delivered in the same room where the corresponding poster will be displayed.



At the start of each session, all authors (four to six per session) should set up their posters. Authors will then give their lightning talks in sequence. Following the talks, authors will remain by their posters to answer questions and engage with delegates.



Below are the templates for the lightning talk and the poster (Size: A1, portrait orientation). 



Please note that all authors are expected to bring a printed copy of their poster.

How to Create QR Code to Link to Digital PDF of Poster





1\) Save the Poster as a PDF.





2\) Copy the PDF file onto a cloud storage platform (e.g. Google Drive, Dropbox, OneDrive)





3\) Open the “Share” menu and select the option “Anyone with the link” as a “Viewer” and copy the link.





4\) Create a QR code from the link copied. For example, you can use an online website - https://www.qrcode-monkey.com/ - which is free to generate a basic QR code to the share link generated previously. There’s a variety of methods to create a QR code from the link.



5\) Download the QR code and insert into the poster and presentation.





6\) Test the QR code to verify that it functions correctly.
